# Financial Sales Report — Power BI

A 3-page Power BI report built on Microsoft's Financials Sample dataset, focused on 2014 sales performance.

## What's Inside
- **Executive Summary** — KPIs and high-level performance overview
- **June Sales Deep Dive** — Analysis of what drove the June growth spike
- **Conclusion & Key Takeaways** — Actionable insights for decision-making

## Technical Highlights
- Redesigned flat dataset into a multi-table model with proper relationships
- DAX measures for sales, units, profit margin, and discount behavior
- Interactive slicers, drill-downs, and custom tooltips

## Tools
Power BI Desktop · DAX · Data Modeling
